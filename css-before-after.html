This is a test of ::before and ::after, with which you can place text or an image before and after each HTML element.

Testsheet:

p.test::before {
	padding-right: 5px;
	content: url(css-before-after_files/logo_ppk.gif);
}
p.test::after {
	font-style: italic;
	content: " and some text after.";
}
