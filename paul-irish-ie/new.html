How it works

Just one <html> tag is hidden; conditional comments surround scripts that each apply a different class to the <html>.

The Advantage

If you need to add junk to your HTML tag – declare a couple namespaces, for example, like the Facebook or Open Graph Protocol namespaces – you can now do so exactly once.

A Slight Disadvantage

Any poor IE users with JavaScript disabled will be treated as if they were using a fully-modern browser.

Of course, this is likely not a problem for many users or for many apps!
