CSS

Cascading Style Sheets

What is it?

Basically CSS is a language used to style or format web pages. It can add color, move or resize page elements, add and control backgrounds, and more. (To see the difference CSS can make, check out the demo.)

CSS is the language no web developer should be without. It was created by the W3C (www.w3.org) to halt HTML's continued bloating. Originally, HTML did not style web pages, but during the "browser wars," it began to include minimal styling attributes. HTML was never able to handle certain layouts. However, CSS gives the web designer near-total control over display of the website, as well as freedom from table-based layouts. Additionally, and to Pixels & Pages the coolest part, is that if your all your web pages reference one CSS file, you can change the way your whole website works by editing that CSS file. That's a whole lot easier and less time-consuming than changing every page by itself!
