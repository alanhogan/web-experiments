Full-Height Body

Most browsers would render this page with a body element with a height of 0 because it has no statically positioned children that would it to grow, but we have added the sinlge CSS declaration html, body { height: 100%; }.

As a result the body is no longer zero-height but should instead match the viewport height. This is a work-around for code that makes faulty assumptions based on the size of the body element.
